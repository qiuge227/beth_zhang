在react之路上一路踩坑

  • 前端 react

自从我们前端组开始使用react做为项目的框架，一路走来确实没有少过躺坑而过的悲惨经历。虽说都是都是正常现象，但是也因此耽误了很多时间来梳理问题原因。

就拿最近我遇到的问题来说：

1.如何使用history来创建router，去掉恶心的＃号的同时，前进后退，使用history自带的方法。
2.如何将ES6完全去转ES5，让浏览器尽数所有的去识别
3.如何优化react组件的问题
4.如何管理config配置文件
5.如何开发前端的单元测试
6.如何结合自动化构建工具，去做项目的稳定性发布
7.如何进行移动端的项目搭建

所有的问题，都在一时间席卷而来。让我这个屌丝前端如何应付的来呢？还好，身边有大神，可以请教。先占一个坑，稍后我再整理。
